# DubSync - Required Icons

This document lists all icons required by the application.
All icons should be SVG files placed in the `icons/` subfolder.

## Naming Convention

- Icon files should use lowercase with underscores (snake_case)
- Each icon should be a single color (recommended: `currentColor` for theme support)
- Recommended size: 24x24 viewBox

## Notes for Designers

1. **Color**: Use `currentColor` in SVG paths to support theming
2. **Stroke Width**: Consistent 1.5-2px stroke width recommended
3. **Style**: Prefer outlined icons for consistency (not filled)
4. **Padding**: Leave ~2px padding around 24x24 icons
5. **Corners**: Use consistent corner radius (2px recommended)
6. **Alignment**: Pixel-align paths for crisp rendering

### Example SVG Template

```xml
<svg xmlns="http://www.w3.org/2000/svg" 
     width="24" height="24" 
     viewBox="0 0 24 24" 
     fill="none" 
     stroke="currentColor" 
     stroke-width="2" 
     stroke-linecap="round" 
     stroke-linejoin="round">
  <!-- paths here -->
</svg>
```
